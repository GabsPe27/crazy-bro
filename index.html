El propósito de la siguiente página es dar a conocer los diversos términos existentes en relación a la informática, tanto en el idioma inglés como en el español, sin más que decir a continuación el glosario de términos / Our purpose in the next page is to make know the many terms existence in relation to the informatic, as much as in english language as well like in spanish; below is show a glossary with informatic terms

"A"

  • Application-Aplicación / "æplɪkeɪʃən" / Example: I found a new application!

  • Antivirus / "ˈæntɪˌvaɪrəs" / Example: She installed an antivirus.

  • Algorithm-Algoritmo / "ælgərɪðəm" / Example: This algorithm is complicated?

B

  • Backup-Copia de seguridad / "bæk ʌp" / Example: You should back up that file.

  • Bandwidth-Ancho de Banda / "bændwɪdθ" / Example: That frees up bandwidth for client.

  • BIOS(Basic Input/Output System)- Sistema básico de entrada y salida / "ˈbaɪˌoʊs" / Example: The BIOS saves the initial computer configuration.

  • Bit-Digito Binario / "bɪt" / Example: A bit is a digit in binary numbers.

  • Bookmark-Marcador / "ˈbʊkˌmɑrk" / Example: I already have a bookmark to google.

  • Boot-Butear / "buːt" / Example: He will boot your computer.

  • Browser-Navegador / "ˈbraʊzər" / Example: You already have that browser?

  • Bug-Error / "bʌg" / Example: Read the bug report for more information.

  • Byte / "baɪt" / Example: A byte is an 8-bit set.

"C"

  • Character- Carácter / "ˈkærəktər" / Example: You can only use 20 characters.

  • Celular- Cellular / "seljʊləʳ" / Example: Please turn off your celullar.

  • Chat / "tʃæt" / Example: Can you answer the chat please?

  • Cyberspace- Ciberespacio / "ˈsaɪbərˌspeɪs" / Example: Do you know something about cyberspace?

  • Command- Comando / "kəˈmænd" / Example: Why the computer doesn't read the command?

  • Controller- Controlador / "kənˈtroʊlər" / Example: You should renew that controller.

  • CPU (Central Processing Unity)- Unidad Central de Procesamiento / "siː piː juː" / Example: Look this new CPU!

D

  • Database-Base de Datos / "deɪtəbeɪs" / Example: Check your database please.

  • Delete- Borrar / "dɪliːt" / Example: Please delete that file.

  • Domain - Dominio / "doʊmeɪn" / Example: To make a website you need a domain.

E

  • E-mail- Correo Electónico / "ˈiˌmeɪl" / Example: Yes, i check your e-mail.

  • Encrypt - Encriptar / "ɪnkrɪpt" / Example: You should encrypt that file.

  • Emulation - Emulación / "ˌɛmjuˈleɪʃən" / Example: An emulator is a software.

F

  • Folder- Carpeta / "foʊldəʳ" / Example: I didn't find that folder.

  • Font- Fuente / "fɒnt" / Example: I like that font.

  • File - Archivo / "faɪl" / Example: You saved the file?

  • Floppy disk- Disquete / "ˈflɒpɪ dɪsk" / Example: Nobody use floppies today.

G

  • Gigabyte / "gɪgəbaɪt" / Example: That hard drive has a capacity of 30 gigabytes.

H

  • Hard disk- Disco Duro / "hɑːd dɪsk" / Example: I bought this new hard disk.

  • Hardware / "hɑːʳdweəʳ" / Example: Hardware is the physical parts of the computer.

  • Hyperlink - link / "lɪŋk" / Example: You check the Hyperlink?

I

  • Icon- Icono / "ˈaɪˌkɑn" / Example: Look the icon that I created.

  • Import- Importar / "ˈɪmˌpɔrt" / Example: Let me import this file.

  • Inbox-Bandeja de Entrada / "ɪnbɒks" / Example: Please check your inbox.

  • Internet / "ɪntəʳnet" / Example: Your internet it's pretty fast.

J

  • Jumper / "dʒʌmpəʳ" / Example: He changed the jumper to complete the circuit.

K

  • Kilobyte / "kɪləbaɪt" / Example: A kilobyte is one thousand bytes.

L

  • LAN / "læn" / Example: The LAN is a computer network.

  • Link-Enlace / "lɪŋk" / Example: You want me to send you the link?

M

  • Megabyte / "megəbaɪt" / Example: A megabyte is one million bytes of data.

  • Modem / "moʊdem" / Example: You checked the modem?

N

  • Network- Red / "ˈnɛtˌwɜrk" / Example: There's not network active.

O

  • Outbox- Buzón de Salida / "ˈaʊtˌbɒks" / Example: Please check your outbox.

  • Open Source- Código Abierto / "ˈoʊpən ˈsɔrs" / Example: Open code is a software development model.

P

  • Password-Contraseña / "pɑːswɜːʳd" / Example: Please enter your new password.

  • Printer- Impresora / "prɪntəʳ" / Example: I bought a new printer.

  • Processor- Procesador / "proʊsesəʳ" / Example : This new processor is very fast.

  • Program- Programa / "proʊgræm" / Example: I installed a new program.

R

  • Router / "ˈraʊtər" / Example: I changed the IP idress from your router.

  • Save- Guardar / "seɪv" / Example: Plase save before turn off your computer.

  • Screen- Pantllas / "skriːn" / Example: My new screen is pretty well.

  • Search Engine-Motor de búsqueda / "sɜːtʃ ˈɛndʒɪn" / Example: The main function of search engines is make more easier our internet navegation.

  • Shortcut-Acceso Directo / "ʃɔrtkʌt" / Example: This shortcut take you to a new window.

  • Turn on- Encender / "tɜːn ɒn" / Example: Press red button to turn on.

  • Turn off- Apagar / "tɜːn ɒf" / Example: Prees green button to turn off.

  • User- Usario / "juːzəʳ" / Example: How many user can play at the same time?

  • Username- Nombre de Usuario / "juːzəʳneɪm" / Example: Please select an Username.

  • Upload- Subir,Cargar / "ʌploʊd" / Example: Wait the upload.

  • Video - Vídeo / "vɪdioʊ" / Example: Wait meanwhile the video loads.

  • Virus / "vaɪərəs" / Example: Searching for virus...

W

  • Website- Sitio Web / "ˈwɛbˌsaɪt" / Example: Look this website!

  • Window- Ventana / "wɪndoʊ" / Example: Open a new window.

  • Wireless- Inlámbrico / "waɪəʳləs" / Example: Wirelees tecnology uses radio waves.

  • Wi-fi / "waɪfaɪ" / Example: There's no wi-fi.
